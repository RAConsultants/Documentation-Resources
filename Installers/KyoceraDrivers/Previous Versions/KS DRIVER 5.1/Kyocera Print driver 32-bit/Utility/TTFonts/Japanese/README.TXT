注意
Windows 3.1, Windows NT3.5xでは、16BitのDirectoryのTrueType Fontをインストールして下さい。
Windows 95/98, Windows NT 4.0,Windows2000では、32BitのDirectoryのTrueType Fontをインストールして下さい。

このディレクトリ内のTrueTypeフォントは、600dpi LSシリーズ専用のスクリーンフォントです。これ以外のプリンタで使用することは、ライセンス上禁止されています。
TrueTypeフォントのインストール方法については、Microsoft Windowsのヘルプを参照ください。
